## 🎮 Usage

### 🖱️ Mouse/Touch Controls

| Button | Function |
|--------|----------|
| `0-9` | Number input |
| `+` `-` `×` `÷` | Mathematical operators |
| `=` | Calculate result |
| `C` | Clear all |
| `CE` | Clear current entry |
| `⌫` | Delete last digit |
| `.` | Decimal point |

### ⌨️ Keyboard Shortcuts

| Key | Function |
|-----|----------|
| `0-9` | Number input |
| `+` `-` `*` `/` | Operators |
| `Enter` or `=` | Calculate |
| `Escape` | Clear all |
| `Backspace` | Delete last digit |
| `.` | Decimal point |

## 🎨 Design Features

### Color Palette
- **Background**: Dark gradient (#0c0c0c → #1a1a2e → #16213e)
- **Operators**: Red gradient (#ff6b6b → #ee5a52)
- **Equals**: Teal gradient (#4ecdc4 → #44a08d)
- **Clear Functions**: Purple gradient (#667eea → #764ba2)
- **Numbers**: Glassmorphic with white text

### Effects
- Backdrop blur for glassmorphism
- Box shadows for depth
- Hover animations with glow
- Smooth transitions
- Ripple effect on click
